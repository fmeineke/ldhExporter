SeekExporter

Simple REST-API for transforming FAIRDOM Seek investigations with NFDI4Health defined MDS extended metadata

Parameter

  • format=[seek|csh|xml|cshval] (Default:csh)
  • id=[investigations|studies|projects]/<number>
All data is retrieved from ${LDH_SOURCE}.

Examples

  • ${LDH_EXP}/export?id=investigations/2
  • ${LDH_EXP}/export?id=investigations/2&format=seek
  • ${LDH_EXP}/export?id=investigations/2&format=csh
  • ${LDH_EXP}/export?id=investigations/2&format=cshval
  • ${LDH_EXP}/export?id=investigations/2&format=xml
