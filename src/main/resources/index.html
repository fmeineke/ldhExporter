SeekExporter

Simple REST-API for transforming FAIRDOM Seek investigations with NFDI4Health defined MDS extended metadata and communicating with Health Study Hub

API

  • /export/id&format=[seek|csh|xml|cshval] (Default:csh)
  • /validate/id
  • /publish/id (requires ${CLIENT_ID} secret)
  • /stats
  • /user list published studies (requires ${CLIENT_ID} secret)
    • GET: id=[investigations|studies|projects]/<number> where json is retrieved from ${LDH_SOURCE}.;

Example

  • GET ${LDH_EXP}/export/projects/2
  • GET ${LDH_EXP}/export/projects/2&format=seek
  • GET ${LDH_EXP}/export/projects/2&format=seekxml
  • GET ${LDH_EXP}/export/projects/2&format=cshxml
  • GET ${LDH_EXP}/export/projects/2&format=csh
  • GET ${LDH_EXP}/export/projects/2&format=xml
  • GET ${LDH_EXP}/validate/projects/2
  • GET ${LDH_EXP}/stats
  • GET ${LDH_EXP}/user
  • POST ${LDH_EXP}/publish/projects/2 (requires ${CLIENT_ID} secret)
